# Cleanfolio

A clean and simple portfolio template built using React, if want a template built with HTML, CSS and JS then check out.



## Instructions

### Setup

```shell
git clone 
cd cleanfolio
```


```shell
nvm install
nvm use
```

Or -

```shell
fnm install
fnm use
```

Install and run -

```shell
yarn
yarn start
```



- Push the changes.

- Build and deploy -

```shell
yarn build
yarn deploy
```
